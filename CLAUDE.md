## 代码文件头部注释规范

所有代码文件（.py / .ts / .vue / .bat 等）必须在文件开头写类似如下注释：

```python
# ============================================================
# backend/main.py
# 一句话描述文件功能
#
# 用法:
#   python main.py
# ============================================================
```

- 注释中包含：文件相对路径、一句话功能描述
- 如果有 CLI 用法（如 Python 脚本），加上"用法"部分
- 注释长度与内容宽度一致（用 `=` 或 `-` 填充到约 60 字符）
