## 代码文件头部注释规范

所有代码文件（.py / .ts / .vue / .bat 等）必须在文件开头写类似如下详细注释：

```python
# ============================================================
# backend/main.py
# 一两句话描述文件功能
#
# 设计与用法:
#   比如导出的函数，导出的类
#   比如导出的api接口
#   等等
# ============================================================
#   列出这个文件的关键常量参数
# ============================================================
```

- 注释中包含：文件相对路径、一句话功能描述
- 如果有 CLI 用法（如 Python 脚本），加上"用法"部分
- 注释长度与内容宽度一致（用 `=` 或 `-` 填充到约 60 字符）
