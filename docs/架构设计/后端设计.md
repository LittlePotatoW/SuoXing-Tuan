## 模块职责

| 模块 | 职责 | 关键文件 |
|------|------|----------|
| `common/` | 共享基础库（数据模型、坐标变换、投影） | schemas.py, transform.py, camera.py, projection.py |
| `inference/` | YOLO AI 推理 | engine.py |
| `fusion/` | 数据融合 + LiDAR-Camera 颜色采样 | datafusion.py, coloring.py, manager.py |
| `reconstruction/` | 三维表面重建 + 缺陷投影 | engine.py, routes.py, projector.py, defect_table.py |
| `state_estimation/` | 自行车模型航迹推算 | estimator.py, router.py |
| `realtime/` | 实时融合调度层 | fusion_router.py, preprocessor.py |
| `loader/` | 离线场景加载器 | scene_loader.py |
| `debug/` | 调试数据采集与回放 | — |

## 数据流

```
location_data → StateEstimator → 位姿 (x, y, yaw)
detection_data → DataFusion → 颜色采样 → ReconstructionEngine → 3D Mesh
                                     ↘ YOLO → DefectProjector → 缺陷标注
```

- 定位数据高频写入（~200ms 间隔），StateEstimator 持续积分航迹
- 检测数据触发式写入，DataFusion 融合点云与图像颜色，送入 ReconstructionEngine
- 每累积 10 帧触发 Poisson 重建，结果通过 WebSocket 推送前端

## 配置

配置文件：`backend/config.yaml`。主要配置项：

- **server**: 监听地址与端口（默认 `127.0.0.1:8000`）
- **model**: YOLO 模型路径、置信度阈值、设备选择（cpu/cuda）
- **sensor**: LiDAR 安装位姿、相机内参/外参标定
- **reconstruction**: Poisson 重建参数（深度、分辨率）
- **state_estimation**: 自行车模型默认轴距
- **fusion**: 颜色采样参数
