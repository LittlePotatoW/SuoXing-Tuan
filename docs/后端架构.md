# SuoXing-Tuan 后端架构

## 目录结构

```
backend/
├── main.py                              # FastAPI 入口 — 挂载所有路由器
│
├── common/                              # 共享基础库，零业务逻辑
│   ├── schemas.py                       # Pydantic 数据模型 (所有 SensorFrame / FusedFrame / MeshData 等)
│   ├── transform.py                     # 坐标变换 + 数学工具 (四元数/旋转矩阵/位姿矩阵，统一来源)
│   ├── camera.py                        # CameraIntrinsics + 内参/外参构建
│   └── projection.py                    # 投影/反投影/桥接函数 (3D→2D, 2D→3D, SensorFrame 投影)
│
├── inference/                           # AI 推理模块 (管线: YOLO 缺陷检测)
│   ├── engine.py                        # InferenceEngine 类
│   └── schemas.py                       # DetectionResult, InferenceResponse
│
├── state_estimation/                    # 航迹推算 (管线: 位姿积分)
│   ├── estimator.py                     # StateEstimator + CarState (自行车模型, 中值滤波, 异常检测)
│   └── router.py                        # /api/preprocessing/* 端点
│
├── fusion/                              # 数据融合 (管线: 多源融合 + 颜色采样)
│   ├── datafusion.py                    # DataFusion: SensorFrame → FusedFrame (坐标变换 + 颜色采样)
│   ├── manager.py                       # DataFusionManager: location + detection → final_data
│   ├── coloring.py                      # 颜色采样 (点云反投影→像素颜色→填充未投影点)
│   └── render.py                        # 可视化渲染 (预留)
│
├── reconstruction/                      # 表面重建 (管线: Poisson 重建 + 缺陷标注)
│   ├── engine.py                        # ReconstructionEngine (Poisson 表面重建 + KDTree 颜色传递)
│   ├── projector.py                     # DefectProjector (2D bbox → 3D 缺陷坐标)
│   ├── defect_table.py                  # DefectTableGenerator + DefectRecord
│   └── routes.py                        # /api/reconstruction/* 端点
│
├── loader/                              # 离线场景加载
│   └── scene_loader.py                  # SceneLoader (PLY + poses.json + images/ → SensorFrame)
│
├── models/                              # AI 模型文件
│   └── crack-detector.pt
│
├── test_data/                           # 测试/场景数据
│   └── test_scene/                      # 30帧合成隧道数据 (PLY + JPEG + poses.json + config.json)                      # 30帧合成隧道数据 (PLY + JPEG + poses.json + config.json)
│
└── tests/                               # 测试
    └── test_pipeline.py                 # 端到端重建管线测试
```

## API 端点

### 主应用 (main.py)

| 方法 | 路径 | 用途 |
|------|------|------|
| GET | `/api/health` | 健康检查 (模型加载状态) |
| POST | `/api/inference` | 单张图片 YOLO 缺陷检测 (multipart) |
| POST | `/api/model/load` | 模型热加载 (.pt/.onnx/.engine) |
| WS | `/ws/realtime` | 实时通信通道 |

### 重建管线 (/api/reconstruction)

| 方法 | 路径 | 用途 |
|------|------|------|
| POST | `/frame` | 上传一帧 SensorFrame (硬件在线模式) |
| GET | `/result` | 获取最新重建结果 (Mesh + 缺陷 + 轨迹) |
| GET | `/status` | 获取引擎状态 |
| POST | `/load` | 加载离线场景文件夹 (回放模式) |
| POST | `/control` | 回放控制 (pause/resume/stop/seek) |
| WS | `/ws` | 实时推送重建进度 |

### 数据预处理 (/api/preprocessing)

| 方法 | 路径 | 用途 |
|------|------|------|
| POST | `/kinematics` | 上传运动学参数 (速度/转向角, 高频) |
| GET | `/position` | 查询指定时刻的小车位姿 |
| GET | `/estimator/stats` | 查询状态估计器统计 |

## 数据流

```
硬件采集
  ├── lacation_data (kinematics, 高频) → StateEstimator → car_position
  │                                                           │
  └── detection_data (点云+图片, 触发) ──→ DataFusionManager ──→ final_data (SensorFrame)
                                                                      │
                                                          POST /api/reconstruction/frame
                                                                      │
                                                                      ▼
                        ┌─────────────────────────────────────────────────┐
                        │  DataFusion.process()                            │
                        │    LiDAR 点云 → 外参变换 → 车体坐标 → 位姿 → 世界坐标    │
                        │    相机图像 → 颜色采样 (反投影+双线性插值+多相机融合)     │
                        │    └─ 同时: YOLO 推理 → 2D bbox → 3D 反投影 → 缺陷标注   │
                        └─────────────────────┬───────────────────────────┘
                                              │
                                              ▼
                        ┌─────────────────────────────────────────────────┐
                        │  ReconstructionEngine                            │
                        │    累积 M 帧 → 体素降采样 → 去离群点 → 法线估计        │
                        │    → Poisson 表面重建 → KDTree 颜色传递 → Mesh       │
                        └─────────────────────┬───────────────────────────┘
                                              │
                                              ▼
                        ┌─────────────────────────────────────────────────┐
                        │  ReconstructionResult                             │
                        │    mesh: MeshData (顶点+面+颜色)                       │
                        │    cracks: CrackAnnotation[] (3D 缺陷位置)              │
                        │    camera_trail: 相机轨迹                               │
                        └─────────────────────────────────────────────────┘
                                              │
                                              ▼
                                    前端 Three.js 渲染
```
