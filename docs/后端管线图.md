## 检测侧 (独立通路)

```
     POST /detection/image          GET /detection/result
              │                            │
              ▼                            ▼
        Detector.detect(image_b64)   Detector.latest()
              │                            │
              ├─ Base64→numpy              └─ 返回 _latest_detections
              ├─ YOLO 推理                    (上次 detect 缓存的结果)
              ├─ 过滤 conf < threshold
              └─ 返回 [{id, class, conf, bbox_2d}, ...]
                       │
                       ▼
              (可选) apply_nms()  去重
              (可选) map_to_3d()  2D框→3D坐标
```

## 完整坐标变换链

```
深度图像素 (u, v, depth_mm)
        │
        ▼  depth_to_pointcloud (Layer 3.5)
相机坐标系 (X右, Y下, Z前)
        │
        ▼  坐标系旋转: Z→X, -X→Y, -Y→Z
标准小车坐标系 (X前, Y左, Z上)
        │
        ▼  camera_to_vehicle 安装外参 (R, T)
实际小车坐标系 (相机安装偏移后)
        │
        ▼  PositionEstimator.get_position_at(ts)
        │  2D 刚体变换: x_w = car_x + X*cos(h) - Y*sin(h)
        │              y_w = car_y + X*sin(h) + Y*cos(h)
        │              z_w = Z
        ▼
世界坐标系 (全局一致)
```
