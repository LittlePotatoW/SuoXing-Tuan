# 位置估计器 — 后端设计文档

## 数据链路

```
POST /api/vehicle/telemetry
  {timestamp, speed, steering_angle}      steering_angle = 当前车轮转角（度），非角速度
        │
        ▼
  vehicle.py 路由 → Pydantic 校验
        │
        ▼
  PositionEstimator.update_telemetry()
    ├─ 自行车模型: speed + 车轮转角 + 轴距 → 更新 (x, y, heading)
    ├─ 追加轨迹点
    └─ 更新 latest_position
        │
        ├──→ GET /api/vehicle/position  ──→  前端查询当前位置
        │
        └──→ engine 内部调用 get_position_at(ts)  ──→  重建时获取帧对应位置
```

一条输入（telemetry），两种读出口（API 查询 / 重建引擎内部调用）。

---

## API 设计

### 遥测上报（写入）

```
POST /api/vehicle/telemetry
```

| 字段 | 类型 | 说明 |
|------|------|------|
| timestamp | float | Unix 时间戳（秒） |
| speed | float | 当前速度 (m/s)，>=0 |
| steering_angle | float | 当前车轮转角（度），正=左转 |

响应: `{"status": "ok"}`

路由逻辑: Pydantic 校验 → `PositionEstimator.instance().update_telemetry(...)` → 返回 ok

### 位置查询（读取）

```
GET /api/vehicle/position
```

响应:

```json
{
  "timestamp": 1715808000.789,
  "x": 10.52,
  "y": 3.18,
  "heading": 45.0
}
```

heading = 航向角度（度），0 = 正前方。路由逻辑: `PositionEstimator.instance().get_position()` → 序列化返回

## Pydantic Schema

```python
# schemas/vehicle.py

from pydantic import BaseModel, Field

class TelemetryRequest(BaseModel):
    timestamp: float
    speed: float = Field(ge=0)
    steering_angle: float          # 度，当前车轮转角

class PositionResponse(BaseModel):
    timestamp: float
    x: float                       # m
    y: float                       # m
    heading: float                 # 度, 0 = 正前方
```

## 核心算法: 自行车模型

steering_angle 是**当前车轮转角**（度），不是角速度。通过自行车模型换算为角速度:

```
给定: speed (m/s), steering_angle (度), wheelbase (m, 轴距)

steering_rad = deg2rad(steering_angle)

if |steering_rad| < 1e-6:
    omega = 0                         # 直线行驶
else:
    turn_radius = wheelbase / tan(steering_rad)
    omega = speed / turn_radius       # rad/s, 角速度

delta_t = timestamp - last_timestamp
heading += rad2deg(omega * delta_t)   # 累计航向角（度）
x += speed * cos(deg2rad(heading)) * delta_t
y += speed * sin(deg2rad(heading)) * delta_t
```

物理含义: 转向角越大 → 转弯半径越小 → 角速度越大。转向角=0 → 直线行驶 → 角速度=0。

## 类接口

```python
from collections import namedtuple

Position = namedtuple('Position', ['timestamp', 'x', 'y', 'heading'])
# heading 单位: 度

class PositionEstimator:
    """位置估计器（单例），自行车模型航位推算"""

    # === 生命周期 ===
    @classmethod
    def create(cls) -> 'PositionEstimator':
        """获取或创建单例实例"""
        ...

    def stop(self) -> None:
        """清空轨迹数据，释放单例"""
        ...

    # === 数据写入 ===
    def update_telemetry(self, timestamp: float,
                         speed: float,
                         steering_angle: float) -> None:
        """接收一条遥测数据，积分更新位置"""
        ...

    # === 位置读取 ===
    def get_position(self) -> Position:
        """返回最新位置"""
        ...

    def get_position_at(self, timestamp: float) -> Position:
        """在轨迹中插值，返回指定时刻的位置。
           用于重建引擎为每帧点云获取精确的采集位置。"""
        ...
```

### 边界情况

| 场景 | 处理 |
|------|------|
| 尚无数据时调用 get_position | 返回 (0, 0, 0, 0) |
| 时间戳回退（新 ts <= 旧 ts） | 丢弃数据，打 warning |
| speed = 0（停车） | 位置和 heading 均不变 |
| steering_angle = 0（直行） | omega = 0，位置沿当前 heading 前进 |
| 长时间无数据后恢复 | delta_t 可能很大，按公式一步积分到位 |

## 模块交互

| 调用方 | 方法 | 场景 |
|--------|------|------|
| routes/vehicle.py | update_telemetry() | telemetry API 到达时 |
| routes/vehicle.py | get_position() | position API 查询时 |
| engine/engine.py | get_position_at(ts) | 重建触发时为每帧获取位置 |

位置估计器不依赖任何其他模块。

## 配置 (config.yaml)

```yaml
# -------------------------------------------
# 位置估计器
# -------------------------------------------
estimation:
  initial_x: 0.0               # 初始 x 坐标 (m)
  initial_y: 0.0               # 初始 y 坐标 (m)
  initial_heading: 0.0         # 初始航向角 (度), 0 = 正前方
  wheelbase: 2.0               # 车辆轴距 (m), 影响转弯半径
  max_history: 10000            # 最大轨迹存储点数, 超出后丢弃旧数据
```
