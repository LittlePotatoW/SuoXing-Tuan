# 项目文件夹结构

## Backend（Python + FastAPI）

已实现 15 个文件，标记 [*]。

```
backend/
├── config.yaml                             # [*] 项目配置（深度相机/重建/估计器/检测）
├── pyproject.toml                          # Python 项目元数据
├── requirements.txt                        # [*] Python 依赖清单
├── server/
│   ├── __init__.py
│   ├── main.py                             # [*] FastAPI 应用入口、CORS、注册路由
│   ├── api/                                # Layer 3 — 后端API层
│   │   ├── __init__.py
│   │   ├── routes/
│   │   │   ├── __init__.py                 # [*] 聚合所有 router
│   │   │   ├── detection.py                # [*] GET /result + POST /image (stub)
│   │   │   ├── reconstruction.py           # [*] GET /status + GET /result，已接入引擎
│   │   │   └── vehicle.py                  # [*] POST /telemetry + /frame + GET /position
│   │   └── schemas/
│   │       ├── __init__.py
│   │       ├── detection.py                # [*] ImageDetectRequest / DetectionItem / DetectionResultResponse
│   │       ├── reconstruction.py           # [*] ReconstructionStatusResponse / ReconstructionResultResponse
│   │       └── vehicle.py                  # [*] TelemetryRequest / PositionResponse / FrameRequest
│   ├── pointcloud/                         # Layer 3.5 — 点云转换层
│   │   ├── __init__.py
│   │   └── converter.py                    # [*] depth_to_pointcloud() 针孔模型深度图→N×3点云
│   ├── engine/                             # Layer 4 — 重建引擎（单例）
│   │   ├── __init__.py
│   │   ├── engine.py                       # [*] ReconstructionEngine: push_frame → 缓存 → 阈值触发 → 重建
│   │   └── frame_buffer.py                 # [*] FrameBuffer: 线程安全帧缓存
│   ├── estimation/                         # Layer 4.5 — 位置估计器（单例）
│   │   ├── __init__.py
│   │   └── estimator.py                    # [*] PositionEstimator: 自行车模型航位推算
│   ├── reconstruction/                     # 三维重建算法 (待实现)
│   │   ├── __init__.py
│   │   ├── mapper.py                       # 点云拼接/网格生成
│   │   └── fusion.py                       # 全量/增量融合
│   ├── detection/                          # YOLO 检测服务 (待实现)
│   │   ├── __init__.py
│   │   ├── detector.py                     # 模型加载与推理
│   │   └── postprocess.py                  # NMS、坐标映射后处理
│   ├── config/
│   │   └── __init__.py                     # [*] YAML 配置加载器
│   └── utils/
│       ├── __init__.py
│       └── timestamp.py                    # 时间戳工具
└── tests/                                  # 测试 (待实现)
    ├── __init__.py
    ├── test_api.py
    ├── test_engine.py
    └── test_estimator.py
```
