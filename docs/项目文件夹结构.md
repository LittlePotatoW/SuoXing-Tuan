## 架构层次总览

| 层号 | 名称 | 归属 | 位置 | 状态 |
|------|------|------|------|------|
| 0 | 网络工具包 | 公共 | `frontend/network/` | ✅ |
| 0 | 外部网络接发 | 前端 | `composables/useConnection.ts` | ✅ |
| 0 | 数据加载 | 前端 | `services/data-loader/` | ✅ |
| 0 | 显示层 | 前端 | `views/` + `components/` | ⚠ 骨架+布局 |
| 0.5 | 数据保存 | 前端 | `services/data-saver/` | ✅ |
| 1 | 包装与解包 | 前端 | `services/pack-unpack/` | ✅ (parse) + 预留 (pack) |
| 1 | 渲染引擎 | 前端 | `services/renderer/` | ❌ 暂缓 |
| 1.5 | 数据预处理 | 前端 | `services/preprocessing/` | ✅ |
| 2 | 前端API层 | 前端 | `api/` | ✅ |
| 3 | 后端API层 | 后端 | `server/api/` | ✅ |
| 3.5 | 点云转换层 | 后端 | `server/pointcloud/` | ✅ |
| 4 | 重建引擎 | 后端 | `server/engine/` | ✅ |
| 4.5 | 位置估计器 | 后端 | `server/estimation/` | ✅ |

后端 12 层全部实现，测试 48 个。前端 7/10 层实现。
