# 后端架构

```
backend/
├── main.py                          # FastAPI 入口
├── common/         共享基础库       schemas, transform, camera, projection
├── inference/      AI推理          YOLO引擎
├── fusion/         数据融合         SensorFrame→FusedFrame, 颜色采样, 融合管理
├── reconstruction/ 三维重建         Poisson重建, 缺陷投影, API路由
├── state_estimation/ 航迹推算      自行车模型StateEstimator + /preprocessing端点
├── realtime/       实时融合        调度层, 接收两路数据→送入管线
├── loader/         离线加载        SceneLoader (PLY+poses)
├── models/         模型文件        .pt (gitignored)
├── test_data/      测试数据        test_scene/, realtime/ (gitignored)
└── tests/          测试
```

## 数据流

```
lacation_data → StateEstimator → 位姿
detection_data → DataFusion → 颜色采样 → ReconstructionEngine → 3D Mesh
                                         ↘ YOLO → DefectProjector → 缺陷标注
```
