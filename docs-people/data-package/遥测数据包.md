# 遥测数据包

发送到位置估计器。

---

## 请求

```
POST /api/vehicle/telemetry
Content-Type: application/json
```

```json
{
  "timestamp": 1715808000.123,
  "speed": 1.5,
  "steering_angle": 15.0
}
```

| 字段 | 类型 | 必填 | 单位 | 说明 |
|------|------|------|------|------|
| timestamp | float | 是 | 秒 | Unix 时间戳 |
| speed | float | 是 | m/s | 当前速度，>=0 |
| steering_angle | float | 是 | 度 | 当前车轮转角，正=左转 |

频率：10 Hz（100ms 间隔）

## 响应

```json
{"status": "ok"}
```

## 示例

```json
// 直行, 2 m/s
{"timestamp": 1.0, "speed": 2.0, "steering_angle": 0.0}

// 左转弯, 1.5 m/s, 车轮左转 15°
{"timestamp": 1.1, "speed": 1.5, "steering_angle": 15.0}

// 右转弯, 1 m/s, 车轮右转 10°
{"timestamp": 1.2, "speed": 1.0, "steering_angle": -10.0}

// 停车
{"timestamp": 1.3, "speed": 0.0, "steering_angle": 0.0}
```

---

## 位置查询

```
GET /api/vehicle/position
```

响应：

```json
{
  "timestamp": 1.2,
  "x": 0.45,
  "y": 0.03,
  "heading": 8.3
}
```

| 字段 | 类型 | 单位 | 说明 |
|------|------|------|------|
| x | float | m | 世界坐标 x |
| y | float | m | 世界坐标 y |
| heading | float | 度 | 航向角，0=正前方 |
